# Small Shop Review Response & Insight Agent

## 项目简介

面向小型门店的评论分析与差评处理 Agent。

## 核心功能

- CSV 上传
- 评论分类
- 情绪分析
- 三大问题聚合
- 差评回复草稿
- 安全检查
- 人工审批
- Dashboard
- Trace & Eval
- Demo Mode

## 技术栈

- Streamlit
- Python
- SQLite
- Pydantic
- OpenAI-compatible API / Ollama
- pandas

📋 CSV 格式要求
必填列：
  • review_text — 评论正文
  • rating — 评分（1-5）
  • date / review_date — 评论时间
可选列：
  • review_id — 评论 ID
  • customer_name — 顾客昵称
  • platform — 来源平台
  • store_id / order_id — 门店或订单标识
编码：推荐 UTF-8；如出现乱码可尝试 GBK。
